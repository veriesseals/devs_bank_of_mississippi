# devs_bank_of_mississippi

Devs Bank of Mississippi

Devs Bank of Mississippi is a full-stack Django web application that simulates core banking functionality.
It demonstrates backend development skills using Python, Django, and SQLite.

FEATURES
- User authentication (register, login, logout)
- Checking and savings accounts
- Deposits, withdrawals, and transfers
- Transaction history tracking
- Secure environment variable configuration

TECH STACK
- Backend: Python, Django
- Database: SQLite
- Frontend: Django Templates
- Version Control: Git & GitHub

INSTALLATION
1. Clone repository
2. Create and activate virtual environment
3. Install requirements
4. Configure environment variables
5. Run migrations
6. Start development server
